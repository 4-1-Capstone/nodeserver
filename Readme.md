### node.JS로 제작한 서버입니다.
- .env는 gitignore로 관리되고 있어요.

### 기능 구현
1. 이미지 반환 플라스크 서버 -> node.JS로 리팩토링 완료
2. 라우터 분리 환경 구축 완료
3. 다른 기능들 추가 예정입니다.

### Commit Rules
- feat: comment
    - 기능 개발, 추가 관련
- fix: comment
    - 버그 수정
- refactor: comment
    - 리팩토링, 불필요한 주석 제거, 개행 수정 등 포함
- test: comment
    - 테스트 및 테스트 코드
- chore: comment
    - 빌드, 설정 파일 등 변경 e.g) gitignore, workflows 등
- docs: comment
    - 문서 관리

예시 : feat/#1-login-jwt
